BobTheBot Docs for Starters

Basic commands

1. ".help" command

Use: shows bot commands via Discord.

Sections: main (default), profile (for economy)

.help {section}

2. ".credits" command

Use: shows the developers of the bot

Alt forms: .creators, .devs, .developers

.credits

3. ".ping" command

Use: check the ping of the bot

.ping

Utility commands (CAN BE GLITCHY)

1. ".say" command

Use: bot will repeat the input you put in the command.

Alt forms: .repeat

.say {message you want to repeat}

Economy commands (.profile)

1. "create" command

Use: start using our economy system by creating a profile!

.profile create

2. "stats" command

Use: view economy stats of yourself or another registered account

.profile stats {mention}

NOTE: USERNAMES & NICKNAMES WITHOUT MENTION WILL NOT WORK.

3. "mine" command

Use: get 1-2 virtual cash for mining

.profile mine

4. "trade" command

Use: give money to a person

.profile trade {money amount} {mention}

NOTE: USERNAMES & NICKNAMES WITHOUT THE MENTION WILL NOT WORK.

5. "bet" command

Use: use this command to gain or lose virtual cash

.profile bet {amount}
